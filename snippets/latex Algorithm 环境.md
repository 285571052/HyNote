# latex Algorithm 环境

```latex
\usepackage{algorithm}
\usepackage{algorithmic}
\begin{algorithm}  
    \caption{A}  
    \label{alg:A}  
    \begin{algorithmic}  
        \STATE {set $r(t)=x(t)$}   
        \REPEAT   
        \STATE set $h(t)=r(t)$   
        \REPEAT  
        \STATE set $h(t)=r(t)$   
        \UNTIL{B}   
        \UNTIL{B}  
    \end{algorithmic}  
\end{algorithm}  
```
