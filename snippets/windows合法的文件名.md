# windows合法的文件名

windwos 文件名 保留字

```
< (less than)
> (greater than)
: (colon)
" (double quote)
/ (forward slash)
\ (backslash)
| (vertical bar or pipe)
? (question mark)
* (asterisk)
```

正则

```
<>:"/\\\|\?\*
```
