# ubuntu 16.04 chrome无法拖拽安装crx插件

ubuntu 16.04 chrome拖拽安装crx插件提示错误时， 可以通过命令行启动chrome， 在启动的时候添加如下参数即可拖拽安装插件

```bash
--enable-easy-off-store-extension-install
```

```bash
google-chrome-stable --enable-easy-off-store-extension-install
```
